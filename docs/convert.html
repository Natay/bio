Format conversions

The bio package may be used to

  • convert data from GenBank to other formats: FASTA, GFF
  • extract only certain sections of data: features by name, coordinate, range

View the JSON representation of the data:

bio ncov | head
[
    {
        "id": "ncov",
        "definition": "Severe acute respiratory syndrome coronavirus 2 isolate Wuhan-Hu-1, complete genome",
        "dblink": [
            "BioProject:PRJNA485481"
        ],
        "locus": "NC_045512",
        "feature_count": 57,
        "origin_len": 29903,

View the JSON representation for a feature:

bio ncov --type CDS --gene S | head 
[
    {
        "id": "YP_009724390.1",
        "name": "YP_009724390.1",
        "start": 21563,
        "end": 25384,
        "type": "CDS",
        "strand": 1,
        "location": [
            [

Extract a partial sequence and change the sequence id

bio ncov --fasta --start 100 --end 200 --seqid foo | head -5
>foo Severe acute respiratory syndrome coronavirus 2 isolate Wuhan-Hu-1, complete genome
CGGCTGCATGCTTAGTGCACTCACGCAGTATAATTAATAACTAATTACTGTCGTTGACAG
GACACGAGTAACTCGTCTATCTTCTGCAGGCTGCTTACGGT

Extract the last ten DNA bases

bio ncov --fasta --start -10 --seqid last | head -5
>last Severe acute respiratory syndrome coronavirus 2 isolate Wuhan-Hu-1, complete genome
AAAAAAAAAA
